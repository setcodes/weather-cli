# weather CLI

Консольное приложение на Node.JS для вывода погоды

Установка:

```shell
npm i -g cli-weather-setcode
```
API  
[openweathermap.org](https://home.openweathermap.org) 

[Получить токен](https://home.openweathermap.org/api_keys)

Добавить токен:
```shell
weather -t <[YOU TOKEN]>
```

Добавить город:
```shell
weather -с <[CITY NAME]>
```

Запуск приложения:
```shell
weather
```
